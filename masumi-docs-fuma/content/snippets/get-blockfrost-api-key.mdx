### Getting a Blockfrost API Key

1. Go to <a href="https://blockfrost.io/" target="_blank" rel="noopener noreferrer">Blockfrost.io</a>
2. Sign up for a free account or log in if you already have one
3. Navigate to your dashboard
4. Create a new project for the Cardano network (mainnet or testnet depending on your needs)
5. Copy your project ID - this is your Blockfrost API key
6. Keep this key secure and never commit it to version control

<Callout type="warning">
  Your Blockfrost API key should be treated as a secret. Store it securely in environment variables or a secure configuration system.
</Callout>
